# LinkO JQuery Plugin#


**LinkO** is a simple Jquery Plugin which will attach links to specific elements

##Version##
0.3.0

##Usage

        $('span').linkO({
                
                href: 'http://test.com',
                text: 'Click me',
                target: '_blank'
                
                
            });
            
            
## Vendors ##
jQuery - [http://jquery.com](http://jquery.com)
